# 亲家直聘 - 设计思路文档

## 选定设计方向：温暖关怀型现代主义

我为这个项目选择了**温暖关怀型现代主义**的设计哲学，以下是核心设计决策：

### 设计哲学
**核心理念**：为中年父母打造一个充满人文关怀、易于理解的平台。设计应该传达"我们理解你的焦虑，我们来帮助你"的温暖感，同时保持现代、专业的视觉形象。

### 核心设计原则
1. **可信度优先**：通过实名认证标签、用户头像、详细资料展示来建立信任感。
2. **简洁易用**：避免复杂的交互，让年长用户能快速上手。
3. **温暖人文**：使用柔和的色彩、充足的留白、人性化的文案来传达关怀。
4. **视觉层级**：清晰的信息优先级，重要信息突出展示。

### 色彩哲学
- **主色**：暖橙色 (`#FF8C42`) - 代表温暖、希望、家庭感
- **辅助色**：清爽蓝 (`#4A90E2`) - 代表信任、专业、安全感
- **背景**：柔和米白 (`#FAFAF8`) - 温暖、舒适、不刺眼
- **文字**：深灰 (`#2C3E50`) - 易读、稳重、可信
- **强调**：绿色 (`#52C41A`) - 成功、确认、积极

### 布局范式
- **卡片流设计**：使用卡片展示每个子女资料，卡片间有充足的间距和阴影，营造立体感。
- **非对称布局**：首页采用不对称的卡片排列，避免单调的网格。
- **底部导航**：移动端采用底部标签栏，方便单手操作。

### 签名元素
1. **温暖阴影**：使用柔和的阴影效果（而非硬边框）来区分卡片。
2. **圆角设计**：所有卡片、按钮采用中等圆角（12-16px），传达友好感。
3. **微妙动画**：卡片加载时的淡入效果、按钮的轻微缩放反馈。

### 交互哲学
- **渐进式披露**：不一次性展示所有信息，通过点击卡片逐步展开详情。
- **明确的行动按钮**："查看详情"、"申请联系"等按钮清晰可见，鼓励用户行动。
- **反馈机制**：每个操作都有清晰的视觉反馈（如"已申请"状态变化）。

### 动画指南
- **页面过渡**：淡入淡出效果（200ms）
- **卡片加载**：从下往上滑入（300ms）
- **按钮交互**：轻微缩放（1.02x）+ 阴影加深
- **成功提示**：绿色勾号 + 文字提示

### 排版系统
- **标题字体**：思源黑体 Bold（用于大标题、卡片标题）
- **正文字体**：思源黑体 Regular（用于描述文本）
- **强调文字**：思源黑体 SemiBold（用于关键信息）
- **字体大小层级**：
  - H1: 28px（页面标题）
  - H2: 20px（卡片标题）
  - Body: 14px（描述文本）
  - Small: 12px（辅助信息）

## 实现细节

### 页面结构
1. **首页**：推荐卡片流 + 底部导航
2. **详情页**：完整资料展示 + 申请联系按钮
3. **发布页**：表单填写（简化版，仅包含必填项）
4. **个人中心**：用户信息管理 + 已联系列表

### 色彩代码参考
```css
--primary: #FF8C42;        /* 暖橙色 */
--secondary: #4A90E2;      /* 清爽蓝 */
--background: #FAFAF8;     /* 柔和米白 */
--text-primary: #2C3E50;   /* 深灰 */
--success: #52C41A;        /* 绿色 */
--border: #E8E8E6;         /* 浅灰边框 */
```

这个设计方向既能传达专业性和可信度，又能通过温暖的色彩和人文的排版传达关怀感，特别适合面向中年父母的应用。
